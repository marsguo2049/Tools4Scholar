# Tools4Scholar - 学术研究工具指南

**面向学者的实用工具集合**  
本指南整理了学术研究中常用的工具和资源，涵盖数学计算、文献管理、图表绘制、论文写作等各个环节，帮助提高研究效率。

---

## 📚 文献管理工具

### EndNote
专业的文献管理软件，支持多种引用格式，便于管理大量文献和自动生成参考文献。

- **下载地址**: http://dbnav.lib.must.edu.mo/content/endnote-desktop

### Citavi
功能强大的文献管理和知识组织工具，特别适合与LaTeX配合使用。

---

## 🔢 数学计算与符号运算

### Mathematica
业界领先的数学计算软件，适用于符号计算、数值分析和可视化。

### SymPy
Python的符号数学库，免费开源，适合程序化处理数学问题。

- **文档地址**: https://docs.sympy.org/latest/install.html

### Mathpix Snip
通过截图识别数学公式并转换为LaTeX代码，极大提升公式录入效率。

- **官网**: https://mathpix.com
- **推荐文章**: [免费数学神器！再复杂的公式，只要有照片就能转成LaTeX](https://mp.weixin.qq.com/s/UFfBoxQwsB_GUqTDRv3Asg)

## 📐 绘图与可视化

### GeoGebra
交互式几何、代数、统计和微积分工具，适合数学建模和可视化。

- **下载地址**: https://www.geogebra.org/download
- **案例资源**: https://www.geogebra.org/materials
- **帮助论坛**: https://help.geogebra.org/

### Draw.io (Diagrams.net)
免费在线流程图和图表绘制工具，支持多种导出格式。

- **在线使用**: https://app.diagrams.net

## 📝 LaTeX 写作工具链

### Overleaf
在线LaTeX编辑器，支持多人协作，无需本地安装。

- **学习资源**: https://www.overleaf.com/learn

### Pandoc
强大的文档格式转换工具，支持Markdown、LaTeX、Word等多种格式互转。

- **官网**: https://pandoc.org

### Tables Generator
在线表格生成工具，支持CSV到LaTeX等多种格式转换。

- **网址**: https://www.tablesgenerator.com

## ✍️ 英文写作与润色

### Reverso Context
查看单词和短语在真实语境中的用法，寻找准确的同义替换。

- **网址**: https://context.reverso.net/

### QuillBot
AI辅助改写工具，可重组句子结构，避免重复表达。

- **网址**: https://quillbot.com/

### Wordtune
智能句子改写工具，提供多种表达方式供选择。

- **网址**: https://www.wordtune.com/

### Grammarly
专业语法检查工具，提供实时写作建议（推荐购买高级版）。

- **网址**: https://app.grammarly.com/

### Writefull
专为学术写作设计，支持LaTeX编辑器插件，实时语法检查。

- **网址**: https://www.writefull.com

---

## 💡 使用建议

1. **文献管理**: 根据团队协作需求选择EndNote或Citavi
2. **数学公式**: Mathpix可大幅提升LaTeX公式录入效率
3. **写作润色**: 建议组合使用Grammarly（语法）+ QuillBot（改写）+ Reverso（用词）
4. **版本控制**: 使用GitHub管理代码和数据，便于追踪和协作
